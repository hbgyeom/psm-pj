## 대상자 분류 기준
### 변수 설명
- BS3_1 - 현재 일반담배(궐련) 흡연 여부
- BS12_47 - 궐련형 전자담배 현재사용여부
1. 매일피움
2. 가끔피움
3. 과거엔 피웠으나, 현재 피우지 않음
8. 비해당(문항4-②⑧)
9. 모름, 무응답

### 연구군 분류 기준
- 대조군 - 일반담배(궐련)을 매일 피우고(BS3_1=1), 궐련형 전자담배를 피우지 않음(BS12_47=8)
- 처치군 - 일반담배(궐련)을 과거에 피웠으나, 현재 피우지 않으며(BS3_1=3), 궐련형 전자담배를 매일 피움(BS12_47=1)

| 구분 | BS3_1 | BS12_47 |
|:----|:-----:|:-------:|
| 대조군 | 1 | 8 |
| 처치군 | 3 | 1 |
